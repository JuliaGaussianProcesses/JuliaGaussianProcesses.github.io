GPLikelihoods.jl

GPLikelihoods aims at facilitating the use of latent models with non-conjugate likelihoods. The goal is to represent different models such as classification y∼Bernoulli(g(f(x))))y \sim \textrm{Bernoulli}(g(f(x))))y∼Bernoulli(g(f(x)))) or event counting y∼Poisson(g(f(x))){y \sim \textrm{Poisson}(g(f(x)))}y∼Poisson(g(f(x))) given an adapted transformation ggg.

Note that this package is still a WIP
